كيفية استضافة موقع مجاني على Github

تستخدم صفحات GitHub الآن مهام سير عمل GitHub Action القابلة للتخصيص لإنشاء التعليمات البرمجية ونشرها حتى يتمكن المطورون من التحكم في إطار عمل التأليف والنشر. تعد صفحات GitHub خيارًا قويًا لتخزين المحتوى الثابت للأسباب التالية: انه مجانا. يجعل التعاون سهلاً. يمكن لأي شخص فتح طلب سحب لتحديث الموقع. يتزامن المستودع الخاص بك مع أي تغييرات أجريتها على موقعك. بينما تأتي صفحات GitHub مع اسم مجال افتراضي مثل ، https://YOUR_USER_NAME.github.io/ ، فهي تدعم المجالات المخصصة. يستخدم مهام سير عمل GitHub Action القابلة للتخصيص لعمليات الإنشاء والنشر. دعنا نتعلم كيفية استضافة المواقع الثابتة التي تم إنشاؤها باستخدام HTML على صفحات GitHub!

بعد إنشاء HTML وتخزينه في مستودع ، انتقل إلى علامة تبويب الإعدادات لهذا المستودع.

انقر على "الصفحات" في الشريط الجانبي الأيمن
ضمن الإنشاء والنشر ، اختر إجراءات GitHub
سيقترح لك هذا بعض مهام سير العمل بناءً على الكود الموجود في المستودع الخاص بك. يمكنك اختيار ثابت HTML.
يؤدي النقر فوق تكوين إلى سير عمل معد مسبقًا. لا تتردد في مراجعة YAML وتعديله حسب تفضيلاتك والتزام الكود.
في غضون بضع ثوانٍ ، سيبدأ الإجراء الخاص بك في العمل. سيتم إنشاء عنوان URL ونشر موقعك الثابت على صفحات GitHub إذا نجحت.
توجه إلى عنوان URL الخاص بك المسمى yourusername.github.io/your_repo_name للتحقق من موقع الويب المباشر الخاص بك!
